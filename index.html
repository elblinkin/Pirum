PEAR Channel for PHP repositories. Powered by Pirum.

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover elblinkin.github.com/Pirum
Listing available packages: 
pear remote-list -c elblinkin
Installing a package: 
pear install elblinkin/package_name
Installing a specific version/stability: 
pear install elblinkin/package_name-1.0.0
pear install elblinkin/package_name-beta
Receiving updates via a feed: 
http://elblinkin.github.com/Pirum//feed.xml
